# Ortech Talent

Professional website for Ortech Talent - IT staffing, consulting, and executive coaching.

## About

Ortech Talent connects companies with exceptional IT professionals and provides specialized consulting to drive successful technology initiatives. A family business built on real relationships, not algorithms.

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript
- Formspree for contact form

## Local Development

Simply open `index.html` in a browser, or use a local server:

```bash
npx live-server
```

## Deployment

Deployed on Netlify with automatic deployments from the main branch.

## Contact Form Setup

The contact form uses Formspree. To activate:
1. Sign up at https://formspree.io
2. Create a new form
3. Replace `YOUR_FORM_ID` in `contact.html` with your actual form ID
